You are a Senior Software Architect, Senior Next.js Developer, Prisma Expert, and Product Designer.

Your task is to transform an existing e-commerce platform into a Catering Marketplace and Quotation Management System while preserving as much of the existing architecture as possible.

SOURCE PROJECT

Repository:

https://github.com/Kuzma02/Electronics-eCommerce-Shop-With-Admin-Dashboard-NextJS-NodeJS

PROJECT GOAL

Convert the electronics e-commerce application into a catering marketplace where:

• Customers browse catering packages.
• Customers customize packages.
• Dynamic pricing is calculated in real time.
• Customers request quotations.
• Caterers review and approve quotations.
• Customers confirm bookings.
• Payments are collected through M-Pesa and other payment methods.

IMPORTANT

Do NOT rebuild the application from scratch.

Perform a comprehensive system analysis and adopt the existing architecture wherever possible.

The objective is to maximize code reuse while replacing only the business logic that is specific to e-commerce.
